# 🌾 KrishiMitra

**AI-Powered Multilingual Farmer Advisory Platform**

KrishiMitra is a full-stack AI-powered web application that helps Indian farmers make informed agricultural decisions using real-time weather data, location intelligence, soil information, crop suitability analysis, market insights, and multilingual AI assistance.

Instead of relying solely on an LLM for recommendations, the platform uses a rule-based recommendation engine backed by structured agricultural data. AI is responsible for generating easy-to-understand explanations in the farmer's preferred language, reducing hallucinations while maintaining a conversational experience.

## 🧠 Recommendation Engine

Crop recommendations are generated using a deterministic scoring system based on:

* Current weather
* Soil type
* Geographic location
* Farming season
* Previous crop
* Market price

The highest-scoring crops are recommended to the farmer.

The language model does **not** generate recommendations. It explains recommendations generated by the application using verified data.
